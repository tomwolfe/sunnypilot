# Sunnypilot Roadmap

This is the roadmap for the next major sunnypilot releases. Also check out

* [Milestones](https://github.com/sunnyhaibin/sunnypilot/milestones) for minor releases
* [Projects](https://github.com/sunnyhaibin/sunnypilot/projects?query=is%3Aopen) for shorter-term projects not tied to releases
* [Bounties](https://sunnypilot.ai/bounties) for paid individual issues
* [#current-projects](https://discord.com/channels/469524606043160576/1249579909739708446) in Discord for discussion on work-in-progress projects

## sunnypilot 0.10

sunnypilot 0.10 will be the first release with enhanced validation and safety systems.

* Enhanced validation metrics publisher and consumer system
* ARM NEON optimization for ARM64 processors
* Advanced UI components with Raylib integration
* Real-time system monitoring with thermal management
* Predictive planning with behavior prediction
* Safety supervisor with redundant validation systems
* Model efficiency optimizations (quantization and pruning)
* Comprehensive performance validation tools
* Hardware constraint validation for Comma Three
* `staging` branch with full integration testing

## sunnypilot 0.11

sunnypilot 0.11 will focus on advanced perception and planning capabilities.

* Enhanced multi-camera fusion with 8-camera simulation
* YOLOv8 integration for improved object detection
* Advanced Kalman filter implementation for object tracking
* Predictive safety validation with situation awareness
* Dynamic adaptation based on thermal and resource constraints
* Enhanced navigation with route planning and turn-by-turn guidance
* Advanced perception visualization with 3D tracking
* Simulation framework for comprehensive testing
* Memory optimization with pooling and management
* Power optimization for ARM processors

## sunnypilot 1.0

sunnypilot 1.0 will feature a fully optimized and validated autonomous driving system.

* End-to-end optimized driving system with ARM NEON acceleration
* Advanced safety validation with predictive risk assessment
* Comprehensive system monitoring and diagnostics
* Adaptive control based on environmental conditions
* Enhanced perception with depth estimation and feature extraction
* Real-time performance optimization and scaling
* Advanced UI with predictive planning visualization
* Fully integrated thermal and power management
* Advanced fusion system with multi-camera object tracking
* Enhanced validation and safety metrics publisher/consumer
